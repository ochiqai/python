# Git Bash
#### Git Bash nima ?
* Git Bash - bu Microsoft Windows operatsion tizimlari uchun mo'ljallangan dastur bo'lib, 
u Unix -ga asoslangan qobiq yordamchi dasturlari va Git buyruqlar satri buyruqlari tajribasini ta'minlaydi.
